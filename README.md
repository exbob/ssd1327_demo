# ssd1327_demo

Linux 系统下，使用 I2C 接口，读写 SSD1327 OLED 屏的例程。

微雪电子，1.5 寸 OLED 模块：

* 驱动芯片：SSD1327
* 支持接口：SPI / I2C
* 显示颜色：白色
* 灰度等级：16级
* 高分辨率：128×128（市面同尺寸多为128×64）
* 可视角度：大于160°
* 工作电压：3.3V / 5V
* 模块大小：44.5 x 37（mm）
* 官网资料：https://www.waveshare.net/shop/1.5inch-OLED-Module.htm
